# analog-clock-widget

Minimalistic analog clock widget.<br>

   right click opens a menu<br>
                                 <br>Zmniejsz wielkość      -> Lower the size
                                 <br>Zwiększ wielkość       -> Increase the size
                                 <br>Odwróc kolory          -> Reverse colors
                                 <br>Zakończ                -> Terminate
