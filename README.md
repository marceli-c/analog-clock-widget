# analog-clock-widget

<br>Minimalistic analog clock widget.
<br>right click opens a menu<br>
                                      Zmniejsz wielkość      -> Lower the size
                                      Zwiększ wielkość       -> Increase the size
                                      Odwróc kolory          -> Reverse colors
                                      Zakończ                -> Terminate
